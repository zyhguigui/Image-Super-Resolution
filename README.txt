# Image-Super-Resolution

1. Convolution Neural Network

Matlab demo code for "Learning a Deep Convolutional Network for Image Super-Resolution" (ECCV 2014) and "Image Super-Resolution Using Deep Convolutional Networks" (arXiv:1501.00092).

2. Sparse Coding (General)

Matlab demo code for "J. Yang et al. Image super-resolution via sparse representation". IEEE Transactions on Image Processing.

3. Sparse Coding (MRI)

Matlab code adapted from "Sparse Coding (General)" for MRI data super resolution.
